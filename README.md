Vous aurez ensuite le choix entre 3 modes :

Un mode joueur contre joueur (2 joueurs), où vous pourrez jouer à la fois les coups de blancs et des noirs

Un mode joueur contre IA (Solo), où vous pourrez jouer contre une ia de niveau environ 1900 elo.

Vous pourrez dans ce mode choisir de jouer les blancs où les noirs

Un mode IA contre IA, où vous pourrez regarder 2 IA qui s'affrontent
